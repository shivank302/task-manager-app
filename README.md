# 📋 Task Manager Web App

A simple yet powerful **Task Manager Web Application** built using **HTML, CSS, and JavaScript** to help users organize and track their daily tasks efficiently.

---

## 🌐 Live Demo

👉 https://shivank302.github.io/task-manager-app/

---

## ⚡ Features

✨ Add tasks with title, description, and due date
✨ Mark tasks as completed ✔
✨ Edit tasks anytime ✏
✨ Delete tasks 🗑
✨ Filter tasks (All / Completed / Pending)
✨ Smart reminder alerts for upcoming deadlines ⏰
✨ Data persistence using Local Storage
✨ Clean and responsive UI

---

## 🧠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API

---

## 🎮 How It Works

1. Enter task title and description
2. Select a due date 📅
3. Click **Add Task**
4. Manage tasks using action buttons
5. Use filters to organize your tasks
